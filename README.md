# 🚴 MyBike Landing Page

A responsive landing page for a fictional bicycle brand built from a Figma design. The project showcases modern frontend development practices with smooth navigation, adaptive layouts, and clean UI implementation.

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/VitaliyHoroshko/MyBike-Landing-Page.git
cd mybike
```

### 2. Install dependencies

```bash
npm install
```

or

```bash
yarn install
```

### 3. Start the development server

```bash
npm start
```

or

```bash
yarn start
```

### 4. Build the project

```bash
npm run build
```
